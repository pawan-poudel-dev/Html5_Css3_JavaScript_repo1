EcoMart is a front-end e-commerce website designed to promote eco-friendly and sustainable lifestyle products.
The goal of this project is to encourage environmentally responsible shopping by providing users with a clean, simple, and interactive online shopping experience.

This project was developed as part of my coursework using HTML, CSS, and JavaScript, focusing on usability, responsiveness, and client-side functionality without any external frameworks.
